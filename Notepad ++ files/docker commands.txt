Docker basic commands

1. docker ps
2. docker ps -a
3. docker -v or docker version
4. docker info ( will work if docker is installed )
5. docker login ( after that command it will ask for login and password to connect hub.docker.com )
6. docker images ( to show all images under docker )
7. docker pull  ( to run the docker image)
8. docker rmi  ( to remove the docker image )
9. docker run ( imagename) ( to run the image ) Once run... run the cmd (docker run -it imagename)
10. docker start ( container ID)( to start the service )
11. docker stop ( container ID )( to stop the docker image )
12. docker stats ( give the details for the running container and image id etc )
13. docker system df
14. docker system prune
